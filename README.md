# Employee Analytics Dashboard

**What it is**
- A responsive web dashboard that displays employee data in an interactive, paginated grid.

**Why use it**
- Centralizes workforce data and enables fast filtering, sorting, and analysis in a single UI.

**Key benefits**
- Fast data exploration with sorting, filtering, pagination, and column sizing.
- Responsive and usable on desktop and mobile.
- Easy to integrate (local JSON or API-backed) and extend.

**Why AG Grid**
- High performance (virtualization for large datasets), rich built-in features (filters, grouping, pagination), and strong customization (cell renderers, formatters).

**Useful for**
- HR analytics, Finance, Sales/CRM, Operations, BI dashboards, and admin portals.

## Quick start

1. Install dependencies

```
cd grid-dashboard
npm install
```

2. Start the dev server

```
npm run dev
```

3. Open http://localhost:5173 (or the URL shown by Vite) and explore the grid.

4. Data source

Employee data is stored in `src/data/employees.json`; swap with an API or different dataset as needed.
